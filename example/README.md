üWave Example
=============

This is an example of a üWave web application using [u-wave-api-v1][api-v1] and
[u-wave-web][web].

Try it by running:

```shell
npm run build
# Pick a port you like! :)
PORT=80 npm start
```

[api-v1]: https://github.com/u-wave/u-wave-api-v1
[web]: https://github.com/u-wave/u-wave-web
